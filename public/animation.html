§ 04 · How to installOne-time setup
Gmail
  1. Open the downloaded HTML file in a browser. The signature renders visually.
  2. Select the entire signature (cmd+A or click-and-drag), copy (cmd+C).
  3. Open Gmail → Settings (gear icon) → See all settings → Signature.
  4. Create or edit a signature, paste (cmd+V) into the editor, save changes.
Outlook desktop
  1. Open the downloaded HTML file in a browser. Select all and copy.
  2. In Outlook: File → Options → Mail → Signatures.
  3. Create or edit a signature, paste, save.
Apple Mail
  1. Mail → Settings → Signatures.
  2. Create a new signature for the account, type any placeholder text.
  3. Open the downloaded HTML file in a browser, select all, copy.
  4. Quit Mail. Open ~/Library/Mail/V[10]/MailData/Signatures/ and find the .mailsignature file you just created (most recent timestamp).
  5. Open it in a text editor, replace the body content with the signature HTML, save.
  6. Restart Mail. Signature is in place.
