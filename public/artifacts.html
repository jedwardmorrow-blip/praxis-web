Install · HTML head

Drop into praxis-web

Replace the existing favicon link tags in app/layout.tsx (or your head component) with these:

<link rel="icon" href="/favicon.svg" type="image/svg+xml" />
<link rel="apple-touch-icon" href="/apple-touch-icon.svg" />

Why two SVGs

Cog teeth disappear at 16px

The full badge has 12 cog teeth that turn to mush below 32px. The simplified variant drops the teeth and thickens the gold ring so the mark stays legible in a browser tab. Modern browsers will pick the right one automatically when both are linked.

Legacy ICO

If you need .ico

Some legacy environments still want a favicon.ico file. Convert favicon-simplified.svg to a 16×16 / 32×32 multi-resolution ICO via realfavicongenerator.net or similar. SVG is the canonical source, ICO is generated.

Install · Gmail

Three steps

  1. Open your .html signature file in a browser
  2. Select all (Cmd-A) and copy (Cmd-C)
  3. Gmail → Settings → General → Signature → paste, save
Install · Apple Mail / Outlook

Same approach

Open the HTML in a browser, select all, copy, paste into the signature field. Apple Mail and Outlook both preserve the inline styles. The badge is inline SVG so no image hosting required and it survives forwarding.

Why system fonts

Big Shoulders not in email

Email clients strip @import. Arial Black approximates Big Shoulders for the name. Courier New approximates Plex Mono for chrome. Georgia italic carries the Special Elite warmth. Close enough; preserves the system in a hostile environment.

Print specs

Standard US 3.5 × 2 in

  1. Trim size: 3.5 in × 2 in (1050×600 px @ 300dpi)
  2. Bleed: 0.125 in on all sides (full SVG = 1125×675)
  3. Safe zone: 0.125 in inside the trim line
  4. Color mode: convert SVG → PDF in CMYK before submitting
Recommended vendors

Print partners

MOO for premium feel (Luxe stock 32pt with painted edge in navy or red). Vistaprint for fast turnaround. Jukebox for letterpress on cotton paper if you want the operator's-manual feel to be physical, not just visual. Recommend MOO Luxe for a first reorder.

Convert to print PDF

SVG → PDF · CMYK

Open the SVG in Affinity Designer or Illustrator, set color profile to CMYK, export as PDF/X-1a with bleed. Most online vendors accept PDF/X-1a directly. Spot color for the red (Pantone 199 C is closest) only if doing letterpress.
